# hubot-wiki

A hubot script for searching wikipedia

See [`src/wiki.coffee`](src/wiki.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-wiki --save`

Then add **hubot-wiki** to your `external-scripts.json`:

```json
["hubot-wiki"]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
